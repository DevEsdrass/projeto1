Principais Tags HTML

h1 ao h6

São tags que na maioria das vezes definem títulos e subtítulos de uma página na web.

'p'

É uma tag que define um parágrafo nesse caso na maioria dos textos se usa a tag 'p' e também alguns atributos como 'strong' 'i' que podem modificar o tipo de texto que é apresentado.

'ol' ou 'ul'

Essas tags você pode escolher de acordo com a necessidade que você vai ter em questão de ser uma lista ordenada ou não para isso se usam as tags 'ol' ou 'ul' seguindo respectivamente de suas 'li'.

'a'

É uma tag que define um link para outra página ou para outro conteúdo dentro da mesma página formando assim um tipo de ancoragem ou ligação com a url ou o arquivo que for linkado ao mesmo.

'blockquote'

Serve como um tipo de Citação de uma frase a seguir.
